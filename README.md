# StoreSmart Shortcode Widgets for Visual Composer

To install:

1. ```git clone``` or download this repo into ```wp-content/themes/storesmart-child```.
2. Add ```require_once('storesmart-vc-templates/vc_map.php');``` to ```functions.php```.
